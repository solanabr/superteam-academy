# CMS Guide

## Sanity Studio
Access at: https://superteam-academy.sanity.studio

## Content Schema
- **Course**: title, slug, description, difficulty, modules, xpReward
- **Module**: title, lessons[]
- **Lesson**: title, type (reading/coding), content (markdown), starterCode, testCases, xpReward

## Creating a Course
1. Open Sanity Studio
2. Click "Course" → "New"
3. Fill in title, slug, difficulty
4. Add modules and lessons
5. Publish

## Publishing Workflow
- Draft → Review → Publish
- Published content is live immediately
