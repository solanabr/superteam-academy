# Security Policy

## Supported Versions

Superteam Academy is an actively developed open source project. We commit to providing security updates for the `latest` stable release across both the Next.js frontend and the Anchor framework Solana program.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Superteam Academy, **please do not open a public issue.** Given the financial and reputational impacts on web3 products, it is critical that vulnerabilities are kept confidential until a patch is released to protect users and their assets.

Instead, please send an e-mail to our maintainers at [exyreams@gmail.com](mailto:exyreams@gmail.com).

We will work to assess your report, patch the vulnerability, and safely release the fix. All security vulnerabilities will be promptly addressed.

Please include the following information in your report:
- Type of issue (e.g., buffer overflow, SQL injection, cross-site scripting, smart contract bug, etc.)
- Full paths of source file(s) related to the manifestation of the issue
- The location of the affected source code (tag/branch/commit or direct URL)
- Any special configuration required to reproduce the issue
- Step-by-step instructions to reproduce the issue
- Proof-of-concept or exploit code (if possible)
- Impact of the issue, including how an attacker might exploit the issue

Thank you for helping us keep Superteam Academy secure.
